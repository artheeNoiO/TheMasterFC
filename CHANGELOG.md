## 0.8.0 (2026-07-02)

### New: 🧭 ตำแหน่งนักเตะละเอียดแบบ FM (14 ตำแหน่ง)

จากเดิม 4 กลุ่มหยาบ (GK/DF/MF/FW) ยกเป็น **14 ตำแหน่งแบบ FM**: GK · DL/DC/DR (แบ็คซ้าย/เซ็นเตอร์/แบ็คขวา) · WBL/WBR (วิงแบ็ค) · DM · ML/MC/MR · AML/AMC/AMR (ปีกซ้าย/หมายเลข 10/ปีกขวา) · ST

- **นักเตะทุกคนมีตำแหน่งธรรมชาติละเอียด** + **ตำแหน่งรองสูงสุด 2 ตำแหน่ง** (เล่นได้ โดนโทษน้อย เพดาน 95% แบบ FM) — นักเตะใหม่สุ่มเกิดพร้อมสเตตเด่นตามตำแหน่ง (ปีกเลี้ยง+เร็ว, เป้ายิง+โหม่ง, DM ตัดบอล ฯลฯ)
- **เซฟเก่าเล่นต่อได้เลย** — ระบบแจกตำแหน่งละเอียดให้นักเตะทุกคนอัตโนมัติจากสเตตเด่น (DF เร็ว+เปิดบอลดี → แบ็ค, ช้าแต่โหม่งแกร่ง → เซ็นเตอร์)
- **ช่องบนสนามทั้ง 8 แผนระบุตำแหน่งจริง** — เช่น 4-2-3-1 = GK, DL, DC, DC, DR, DM, DM, AML, AMC, AMR, ST · 3-5-2 ใช้วิงแบ็ค WBL/WBR
- **โทษยืนผิดตำแหน่งละเอียดขึ้น** — คิดตามระยะห่างแถว+ฝั่ง (MC↔DM โดนนิดเดียว, ST ไปยืน DC โดนหนัก, สลับปีกซ้าย↔ขวาโดนน้อย) โชว์ −X% บนโทเคนเสื้อและตารางแทคติกเหมือนเดิม
- **จัดทีมอัตโนมัติฉลาดขึ้น** — เลือกตัวจริงแบบจับคู่ทีละช่องด้วย rating×ฟอร์ม×ความคุ้นเคยตำแหน่ง (ทีมบอทก็ใช้ระบบเดียวกัน)
- **บทบาท (Role) แยกตามตำแหน่งละเอียด** + บทบาทใหม่ 5 แบบ: แบ็คยืนโซน, เพลย์เมคตัวลึก (DLP), มิดฟิลด์ริมเส้น, ตัดเข้าในยิง (Inside Forward), กองหน้าตัวต่ำ (Shadow Striker)
- **UI ทั้งเกมโชว์ตำแหน่งละเอียด** — ตารางแทคติก (พร้อมตำแหน่งรอง), โทเคนบนสนาม, หน้าสควอด, ตลาด/ประมูล/แมวมอง/อคาเดมี
- ระบบกลุ่มเดิม (โค้ชประจำสาย, บอร์ดซ้อม GK/DF/MF/FW, engine online) ยังทำงานเหมือนเดิมทุกอย่าง

## 0.7.3 (2026-07-02)

### Changed: 📋 หน้าแทคติกโฉมใหม่ (สไตล์ Top Eleven)

**ตารางนักเตะแบบ Top Eleven** — แทนลิสต์บทบาทแบบเดิม:
- แถวละคน: ป้ายตำแหน่งสี (GK เหลือง / DF ฟ้า / MF เขียว / FW แดง / **RES** เทาสำหรับสำรอง) เรียงตัวจริง 11 คนตามช่องบนสนามก่อน ตามด้วยสำรอง
- คอลัมน์: ชื่อ · อายุ · **แถบคอนดิชัน** (เขียว/เหลือง/แดงตามสตามินา) + มูด · OVR · **บทบาท**
- แตะช่องบทบาทเพื่อกางตัวเลือกแล้วเปลี่ยน role ได้จากตารางเลย (✦ = สเตตเหมาะกับบทบาท)
- กดหัวคอลัมน์เรียงได้ (อายุ / คอนดิชัน / OVR) — กดซ้ำสลับทิศ, กดอีกครั้งกลับเรียงตามช่องบนสนาม
- ยืนผิดตำแหน่งมี ⚠️ + บอกโทษ −X% ในตาราง
- ปุ่ม **"จัดทีมอัตโนมัติ"** ท้ายตาราง เลือกตัวจริงที่ดีที่สุดตามแผนปัจจุบันทันที

**การ์ดสไตล์ทีมแบบใหญ่** — จังหวะ / เพรสซิ่ง / แนวรับ เปลี่ยนจากปุ่มเล็กในหน้าเตรียมแมตช์เป็นการ์ดใหญ่โทนน้ำเงิน (แบบการ์ด Stretching/Cardio ใน Top Eleven) พร้อมไอคอน + สรุปผลชัดๆ เช่น "+6% รุก · แรง x1.3" ปรับได้จากแท็บแทคติกโดยตรง มีผลทุกนัด:
- รวมสวิตช์ **กับดักล้ำหน้า** พร้อมโชว์ความเร็วเฉลี่ยกองหลังปัจจุบันประกอบการตัดสินใจ
- หน้าเตรียมแมตช์ยังปรับเฉพาะนัดได้เหมือนเดิม (ค่าเดียวกัน ซิงก์กัน)

**กระดานสนามลากวาง** ยังอยู่ตำแหน่งเดิม — ตาราง/สนาม/การ์ดซิงก์ข้อมูลกันทั้งหมด

## 0.7.1 (2026-07-02)

### New: ⚽ กระดานจัดทีมแบบลากวาง (สไตล์ Top Eleven)

**สนามจัดทีมโฉมใหม่** — แทนที่สนามแบบดูอย่างเดียว + ลิสต์เลือกตัวจริงแบบเก่า:
- **โทเคนเสื้อแข่ง** สีทีมพร้อมเรตติ้ง, วงแหวนตามระดับ (ทอง ≥82 / เงิน ≥72), แถบสตามินา, ป้ายชื่อ + ตำแหน่ง, 🤕 บาดเจ็บ, ⭐ เลเจนด์
- **ม้านั่งสำรอง** เป็นแถวเลื่อนใต้สนาม — ลากขึ้นสนามเพื่อสลับตัว, ลากตัวจริงลงม้านั่งเพื่อถอดออก, ลากสลับช่องกันเองบนสนามได้
- ใช้ pointer events รองรับทั้งเมาส์และจอสัมผัส มี ghost token ตามนิ้วตอนลาก
- โหมดออโต้ = ดูอย่างเดียว (ผจก.จัดให้) — ปิดออโต้เพื่อลากเอง

**วางข้ามตำแหน่งได้ + โทษ out-of-position** — ลากกองกลางไปยืนกองหน้าได้จริงแบบ FM:
- แต่ละตำแหน่งมีความคุ้นเคยต่างกัน เช่น MF ยืน FW/DF เหลือ 85%, FW ยืน DF เหลือ 72%, ใครยืน GK เหลือ 45%
- โทเคนโชว์คำเตือน "ผิดตำแหน่ง −X%" สีแดงชัดเจน
- โทษมีผลจริงทุกโหมด: รายงานก่อนเกม (xG/โอกาสชนะ), แมตช์สด, ข้ามไปจบเกม และซิมทันที

**เปิดได้ 2 ที่**: แท็บแทคติก + หน้าหลักวันแข่ง (แผง "จัดทีมก่อนเกม" กดเปิดสนามได้เลย)

## 0.6.1 (2026-07-02)

### Fix + New: ระบบรวมการ์ดยกเครื่อง

**แก้บัครวมการ์ด** — ยกเครื่องกลไกรวมการ์ดใหม่ทั้งหมด: ตัดสินผลสุ่ม + เลือกใบที่จะใช้ก่อน แล้วค่อยบันทึกผลแบบตายตัว (เดิมสุ่มผลข้างในตัวอัปเดต state ทำให้บางจังหวะผลไม่ตรงกับของจริง/การ์ดไม่ถูกหักออกจากกระเป๋า) พร้อมกันกดปุ่มรวมรัวๆ ไม่ให้รวมซ้ำชุดเดิม

**🤖 รวมการ์ดอัตโนมัติ** — ขี้เกียจนั่งรวมการ์ด tier ต่ำเองไม่ต้องแล้ว:
- เปิดสวิตช์เลือก tier ที่อยากให้รวมอัตโนมัติได้เป็นรายดาว: 1★→2★ / 2★→3★ / 3★→4★ / 4★→5★
- **5★-7★ รวมอัตโนมัติไม่ได้** ต้องกดรวมเองเท่านั้น (ตามที่กำหนด)
- รวมอัตโนมัติทำงาน 2 จังหวะ: ทันทีหลังเปิดการ์ดทุกครั้ง + ปุ่ม "รวมอัตโนมัติตอนนี้" ในหน้ารวมการ์ด
- ระบบรวมซ้ำจนไม่เหลือชุดครบ 10 ใบ — การ์ดใหม่ที่ได้จากการรวม ถ้าสะสมครบ 10 ใบใน tier ที่เปิดไว้ก็รวมต่อให้เลย

**✨ หน้าต่างสรุปผลการรวม** — ทุกการรวม (มือ/อัตโนมัติ) เด้งหน้าต่างสรุปบนหน้าจอ: รวมกี่ชุด สำเร็จ/ไม่สำเร็จกี่ชุด และโชว์การ์ดใบใหม่ที่ได้พร้อมสเตตครบ

## 0.6.0 (2026-07-02)

### New: ระบบแทคติกเชิงลึกแบบ FM

**บทบาทผู้เล่นเฉพาะตำแหน่ง (Player Roles)** — แทนที่ระบบเดิม (รับ/สมดุล/บุก) ด้วยบทบาทแบบ FM จริง 12 แบบ:
- GK: โกลมาตรฐาน / สวีปเปอร์คีปเปอร์ · DF: หลังตัวหยุดเกม / หลังเล่นบอล / แบ็คบุก (Wing-Back)
- MF: แองเคอร์ / บ็อกซ์ทูบ็อกซ์ / เพลย์เมคเกอร์ / วิงเกอร์ · FW: โพชเชอร์ / ตัวเป้า / ฟอลส์ไนน์
- แต่ละบทบาทมีตัวคูณรุก/รับต่างกัน และ**ขึ้นกับสเตตจริงของนักเตะ** — คนที่สเตตตรงบทบาท (มีเครื่องหมาย ✦) ได้โบนัสเต็ม เล่นผิดบทบาทได้ผลน้อยลง
- เซฟเก่า (บทบาท บุก/รับ/สมดุล) แปลงให้อัตโนมัติ ไม่ต้องตั้งใหม่

**จังหวะเกม + การกดดัน (Tempo & Pressing)** — ไดอัลใหม่ในรายงานก่อนเกม:
- Tempo ช้า/ปกติ/เร็ว — เร็วได้พลังรุกแต่เปลืองสตามินามาก (+30%)
- Pressing กดต่ำ/กลาง/สูง — กดสูงบีบคู่แข่งแต่เปลืองแรง +35% และเสี่ยงบาดเจ็บ +30%

**แนวรับ + กับดักล้ำหน้า (Defensive Line & Offside Trap)**:
- ย้าย "แนวรับสูง/ลึก" จากคำสั่งทีมมาเป็นไดอัลแยก 3 ระดับ
- แนวรับสูงเปิดใช้ **กับดักล้ำหน้า** ได้ — ได้ผล (+6% เกมรับ) เฉพาะเมื่อแนวรับเร็วพอ (pace เฉลี่ย ≥13) ถ้าแบ็คช้าโดนบอลทะลุหนักกว่าเดิม (-12%)

**มือดาวเตะลูกตั้งเตะ (Set-Piece Takers)** — เลือกคนเตะคอร์เนอร์/ฟรีคิก/จุดโทษเองในหน้าแทคติก ระบบแนะนำ 5 อันดับตามสเตตจริง (คอร์เนอร์ใช้เปิดบอล+เลี้ยง, ฟรีคิกใช้ยิง+จ่าย, จุดโทษใช้ความนิ่ง+ยิง) — มือเตะดีเพิ่มพลังรุกทั้งเกมสูงสุด +6%

**ประกบตัวอันตราย (Opposition Marking)** — ล็อกประกบนักเตะคู่แข่ง 1 คนจากรายงานก่อนเกม ลดพลังรุกตัวนั้น 5-14% (ยิ่งเป็นตัวหลักยิ่งลดเยอะ) และลดโอกาสเป็นคนยิงประตูลงครึ่งหนึ่ง แลกกับเกมรับเราเสียเล็กน้อย รีเซ็ตหลังจบนัด

**ความคุ้นเคยแทคติก (Tactical Familiarity)** — เหมือน FM จริง:
- ใช้แผนเดิมต่อเนื่องยิ่งนานยิ่งแข็ง: แผนใหม่ -10% → ครบ 15 นัด +5%
- เปลี่ยนฟอร์เมชันเมื่อไหร่เริ่มนับใหม่ทันที มีแถบความคืบหน้าโชว์ในหน้าแทคติกและรายงานก่อนเกม
